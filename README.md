# 图片转文字工具

一个基于 Web 的 OCR（光学字符识别）工具，可以将图片中的文字提取出来。

## 功能特点

- 📷 支持拖拽或点击上传图片
- 🔍 自动识别图片中的中文和英文文字
- 📋 一键复制识别结果
- 🎨 现代化的用户界面
- 📱 响应式设计，支持移动设备

## 使用方法

### 方式一：使用 Node.js 服务器（推荐）

1. 安装依赖：
```bash
npm install
```

2. 启动服务器：
```bash
npm start
```

3. 在浏览器中访问：`http://localhost:3000`

### 方式二：直接打开文件

1. 直接在浏览器中打开 `index.html` 文件
2. 点击上传区域或拖拽图片到页面上
3. 点击"开始识别"按钮
4. 等待识别完成，结果会显示在下方
5. 点击"复制文字"按钮可以复制识别结果

## 技术栈

- **Node.js + Express**: 提供本地服务器
- **Tesseract.js**: 用于 OCR 文字识别
- **纯前端实现**: 无需后端服务器，所有处理在浏览器中完成
- **支持语言**: 中文简体 + 英文

## 浏览器支持

- Chrome/Edge (推荐)
- Firefox
- Safari

## 注意事项

- 首次使用需要下载 OCR 模型文件，可能需要一些时间
- 识别准确度取决于图片质量和文字清晰度
- 建议使用清晰、对比度高的图片以获得最佳效果
